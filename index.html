⚙️ 云端同步配置

1. 打开 jsonbin.io 注册免费账号
2. 左侧 API Keys → 复制 Master Key
3. 左侧 Bins → Create a Bin → 输入 {"purchase":[],"domestic":[],"international":[]} → Save
4. 复制 Bin ID（右侧面板显示）
